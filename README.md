# tyom/skills

Tyom's personal collection of agent skills, installable with the
[skills.sh](https://skills.sh) CLI.

## Install

```bash
npx skills add tyom/skills
```

## Skills

| Skill       | Description                                                                                       |
| ----------- | ------------------------------------------------------------------------------------------------- |
| `explainer` | Builds a self-contained, interactive HTML explainer for any subject, grounded in the real source. |
